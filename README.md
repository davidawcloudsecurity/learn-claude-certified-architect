## Claude Certified Architect (CCA) — Key Points

**What it is:** Anthropic's first official technical credential, launched March 12, 2026. A proctored, architecture-level exam (not a prompting quiz or completion badge).

**Exam format:** 60 questions across 5 domains:

| Domain | Weight |
|--------|--------|
| Agentic Architecture & Orchestration | 27% |
| Claude Code Configuration & Workflows | 20% |
| Prompt Engineering & Structured Output | 20% |
| Tool Design & MCP Integration | 18% |
| Context Management & Reliability | 15% |

**Access:** Currently requires joining the Claude Partner Network (free for any organization bringing Claude to market). First 5,000 partner employees get early access at no cost.

**Preparation:** 13 free courses on [Anthropic Academy (Skilljar)](https://anthropic.skilljar.com/) — open to everyone regardless of partner status. Key courses:
- Claude 101
- AI Fluency Framework
- Building Applications with Claude API
- Introduction to Model Context Protocol (MCP)
- Claude Code developer training

**Timeline:**
- Experienced AI developers: 6–10 weeks
- Newer developers: 2–4 months

**Recommended projects before the exam:**
1. Claude RAG knowledge assistant
2. AI support chatbot with tool use
3. Multi-step tool-enabled agent workflow
4. Document analysis assistant
5. Enterprise AI automation workflow

**Skills expected:** Prompt engineering, Claude API integration, RAG systems, tool use / agent workflows, AI system architecture, Python, and at least one production Claude deployment.

**Career relevance:** Major firms (Accenture ~30K, Cognizant ~350K employees) are training on Claude. Roles it maps to: AI solutions architect, LLM application architect, AI platform engineer, AI developer.

**Coming later in 2026:** Developer certification and advanced architect tiers.

---

The bottom line: it's a systems-design exam, not a trivia test. Build real projects first, then certify. The free Anthropic Academy courses cover all five domains.
